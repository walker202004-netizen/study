# NotebookLM 提示词库

Google NotebookLM 是一个由AI驱动的研究助手，可以帮助用户快速从各种信息源中进行研究、学习和创建内容。本文档收集了针对 NotebookLM 优化的高质量提示词和使用技巧。

## 🎯 提示词分类

### 📚 研究相关
- 文献综述提示词
- 研究问题框架
- 数据收集和整理

### 📖 学习与笔记
- 知识总结与提炼
- 笔记组织和管理
- 学习路径规划

### ✍️ 内容创作
- 文章生成
- 演讲稿撰写
- 报告生成

### 🔍 分析与总结
- 文档分析
- 关键信息提取
- 对比分析

## 💡 NotebookLM 使用技巧

1. **上传多个来源** - 可以同时上传多个文档或网页，NotebookLM会进行综合分析
2. **细化问题** - 提问时要具体，这样会获得更精准的答案
3. **引用追踪** - NotebookLM会标注信息出处，便于验证
4. **迭代优化** - 根据初始结果继续提问和优化

## 🚀 快速开始

1. 访问 [NotebookLM](https://notebookllm.google.com/)
2. 创建新 Notebook
3. 上传相关文档或资源
4. 使用本文档中的提示词提问和分析
5. 根据需要调整和优化提示词

## 📝 提示词模板

在下方添加具体的提示词模板：

### 研究综述

**用途**：快速了解某个主题的全面概览

**提示词**：
```
请根据上传的文档，为我总结 [主题] 的主要内容，包括：
1. 核心概念
2. 关键观点
3. 主要发现
4. 未来方向

请标注每个观点的来源。
```

### 对比分析

**用途**：对比不同观点或方法的优缺点

**提示词**：
```
请对比文档中关于 [话题] 的不同观点，列出：
1. 每个观点的主要内容
2. 支持该观点的证据
3. 潜在的限制或反驳
4. 我应该如何理解它们之间的差异
```

### 学习路径规划

**用途**：制定系统的学习计划

**提示词**：
```
根据上传的资源，为我设计一个学习 [主题] 的路径：
1. 应该按什么顺序学习
2. 每个阶段的学习目标
3. 如何巩固理解
4. 如何评估学习效果
```
