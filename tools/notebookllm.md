# NotebookLM 提示词库

Google NotebookLM 是一个由AI驱动的研究助手，可以帮助用户快速从各种信息源中进行研究、学习和创建内容。本文档收集了针对 NotebookLM 优化的高质量提示词和使用技巧。

## 🎯 提示词分类

### 📚 研究相关
- 文献综述提示词
- 研究问题框架
- 数据收集和整理

### 📖 学习与笔记
- 知识总结与提炼
- 笔记组织和管理
- 学习路径规划

### ✍️ 内容创作
- 文章生成
- 演讲稿撰写
- 报告生成

### 🔍 分析与总结
- 文档分析
- 关键信息提取
- 对比分析

## 💡 NotebookLM 使用技巧

1. **上传多个来源** - 可以同时上传多个文档或网页，NotebookLM会进行综合分析
2. **细化问题** - 提问时要具体，这样会获得更精准的答案
3. **引用追踪** - NotebookLM会标注信息出处，便于验证
4. **迭代优化** - 根据初始结果继续提问和优化

## 🚀 快速开始

1. 访问 [NotebookLM](https://notebookllm.google.com/)
2. 创建新 Notebook
3. 上传相关文档或资源
4. 使用本文档中的提示词提问和分析
5. 根据需要调整和优化提示词

## 📝 提示词模板

在下方添加具体的提示词模板：

### 研究综述

**用途**：快速了解某个主题的全面概览

**提示词**：
```
请根据上传的文档，为我总结 [主题] 的主要内容，包括：
1. 核心概念
2. 关键观点
3. 主要发现
4. 未来方向

请标注每个观点的来源。
```

### 对比分析

**用途**：对比不同观点或方法的优缺点

**提示词**：
```
请对比文档中关于 [话题] 的不同观点，列出：
1. 每个观点的主要内容
2. 支持该观点的证据
3. 潜在的限制或反驳
4. 我应该如何理解它们之间的差异
```

### 学习路径规划

**用途**：制定系统的学习计划

**提示词**：
```
根据上传的资源，为我设计一个学习 [主题] 的路径：
1. 应该按什么顺序学习
2. 每个阶段的学习目标
3. 如何巩固理解
4. 如何评估学习效果
```

### 构建专业知识库 - 资源搜寻与分类

**用途**：构建特定领域的专业知识库，进行多维度、分级别的学习资源搜寻

**提示词**：
```
我正在使用 NotebookLM 构建 [领域] 专家知识库。请帮我在 YouTube 上搜寻 8-10 个高质量视频，要求如下：

【覆盖三个维度】：
1. 基础架构：包含 [基础内容示例，如安装、核心概念、基本操作]
2. 高阶逻辑：包含 [进阶内容示例，如复杂功能、代码集成、错误处理]
3. 实战案例：包含 [实战应用示例，如具体项目、多系统集成、复杂工作流]

【筛选标准】：
1. 优先选择官方频道或公认专家的视频
2. 必须有清晰的口述讲解，避免纯录屏无解说
3. 寻找包含大量"逻辑解释"而非仅仅"鼠标操作"的视频

【输出格式】：
对于每个视频，请列出：
- 标题与链接（附带发布年月）
- 难度等级（初级/中级/专家）
- 关键节点/功能清单（列出该视频中重点演示的 3-5 个核心要素）
- 核心逻辑总结（描述该视频解决的核心问题）
```

### 内容综合总结

**用途**：基于多个信息源生成完整而凝练的总结

**提示词**：
```
请基于我上传的内容以及我们当前的对话，生成一份完整而凝练的总结，要求如下：

1. 保留所有关键概念、事件、人物、数据、结论和逻辑关系
2. 对话中出现的有价值问答或观点也要整合进总结
3. 不要省略任何信息，即使内容有重复也请保留
4. 输出清晰、按逻辑顺序排列，便于理解
5. 可以使用概览 → 详细内容的方式呈现

【输出结构】：
- 概览：核心主题和关键信息的高层概括
- 详细内容：按逻辑分块呈现所有重要信息
- 一句话总结：概括内容的核心主题或结论

请直接生成总结，不需要额外解释。
```
