## 💡 NotebookLM 使用技巧

1. **上传多个来源** - 可以同时上传多个文档或网页，NotebookLM会进行综合分析
2. **细化问题** - 提问时要具体，这样会获得更精准的答案
3. **引用追踪** - NotebookLM会标注信息出处，便于验证
4. **迭代优化** - 根据初始结果继续提问和优化

## 🚀 快速开始

1. 访问 [NotebookLM](https://notebookllm.google.com/)
2. 创建新 Notebook
3. 上传相关文档或资源
4. 使用此库中的提示词提问和分析
5. 根据需要调整和优化提示词
