# Study - 学习工具提示词库

一个专业的提示词（Prompt）集合仓库，汇集了各大学习工具和AI助手的最佳实践提示词，以及通用的学习与工作提示词模板。

## 📋 项目简介

本仓库致力于收集、整理和分享高质量的提示词资源，帮助用户更高效地利用各类学习工具和AI助手。无论您是学生、教师、研究者还是专业人士，都能在这里找到适合自己的提示词模板。

## 🎯 提示词分类

### 📚 通用提示词
- **写作提示词** - 论文、文案、创意写作等
- **学习提示词** - 知识总结、笔记整理、理解辅助等
- **头脑风暴** - 创意生成、问题分析等
- **效率提升** - 时间管理、任务规划等

### 💻 AI 工具提示词
- **ChatGPT** - 针对 OpenAI ChatGPT 优化的提示词
- **GitHub Copilot** - 代码生成和编程辅助
- **Claude** - Anthropic Claude 专用提示词
- 其他 AI 工具 - 如 Gemini、Perplexity 等

### 🔧 编程学习
- 各编程语言的学习提示词
- 算法与数据结构
- 最佳实践与代码审查

### 🌍 语言学习
- 英语学习提示词
- 中文学习提示词
- 其他外语学习资源

## 🚀 使用方法

1. **浏览仓库** - 根据您的需求找到对应分类的提示词
2. **复制提示词** - 直接复制使用，或根据具体情况修改
3. **分享反馈** - 发现更好的提示词？欢迎贡献！

## 💡 提示词使用建议

- **个性化调整** - 根据实际需求修改提示词中的关键词和参数
- **组合使用** - 多个提示词可以组合使用以获得更好效果
- **版本迭代** - 不断测试和改进提示词，找到最适合的版本
- **记录反馈** - 记下每个提示词的效果，便于后续优化

## 🤝 贡献指南

欢迎各位贡献优秀的提示词！

### 如何贡献

1. Fork 本仓库
2. 创建新分支 `git checkout -b feature/add-new-prompts`
3. 添加或改进提示词，确保格式规范
4. 提交 Pull Request，附上详细说明

### 贡献标准

- 提示词应清晰、实用、经过测试
- 提交时请包含：
  - 提示词的用途说明
  - 适用场景
  - 使用示例（可选）
- 遵循现有的目录结构和命名规范

## 📝 提示词模板格式

建议的提示词文件格式：

```markdown
# 提示词标题

**用途**：简要说明此提示词的主要用途

**适用工具**：ChatGPT / Claude / Copilot 等

**适用场景**：描述最适合使用此提示词的情况

## 内容

[具体的提示词内容]

## 使用示例

[使用示例或输出示例]

## 优化建议

[可选的优化方向或变体]
```
